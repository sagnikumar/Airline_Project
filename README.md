# Airline Passenger Satisfaction Prediction

## Overview

This project analyzes airline passenger data and builds a machine learning model to predict passenger satisfaction. The goal is to identify key factors that influence customer experience and provide insights that can help airlines improve their services.

---

## Problem Statement

Airlines need to understand what drives passenger satisfaction. This project aims to analyze customer data and predict whether a passenger is satisfied or not based on various features such as service quality, delays, and travel class.

---

## Dataset

* Airline passenger dataset
* Includes features like:

  * Gender, Age
  * Customer Type
  * Type of Travel
  * Class (Economy/Business)
  * Flight Distance
  * In-flight services (seat comfort, food, cleanliness, etc.)
* Target Variable:

  * **Satisfaction** (Satisfied / Neutral / Dissatisfied)
