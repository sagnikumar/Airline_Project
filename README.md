# Airline Passenger Satisfaction Prediction

## Overview

This project focuses on analyzing airline passenger data and building a machine learning model to predict passenger satisfaction. It includes data cleaning, exploratory analysis, and model development.

## Problem Statement

Airlines want to understand what factors influence passenger satisfaction. This project analyzes customer data and predicts whether a passenger is satisfied or not.

## Dataset

* Airline passenger dataset (features like age, travel class, delay, service ratings, etc.)
* Target variable: Satisfaction (Satisfied / Neutral / Dissatisfied)

## Tools & Technologies

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* SQL (for data analysis)

## Project Workflow

1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Building (e.g., Logistic Regression / Random Forest / XGBoost)
5. Model Evaluation

## Results

* Built a classification model to predict passenger satisfaction
* Achieved good accuracy and performance on test data
* Identified key factors affecting satisfaction

## How to Run

1. Clone or download this repository
2. Open the Jupyter Notebook
3. Run all cells step by step
